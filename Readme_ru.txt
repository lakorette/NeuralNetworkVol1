
------------------------------------------------------------------------
Файлы и папки:
Main.py - главный файл(методы интерфейса, связь интерфейса с другими файлами)
MainWindow.py - интерфейс главного окна
CreationWindow.py - интерфейс окна создания нейросети
neuralnetworks.py - файл с классом нейросети
settings.py - файл с классом настроек
---
\Data\Networks - сохраненные нейронные сети
\Data\Query - картинки для тестирования(кнопка "open query file")
\Data\Test - данные для тестирования нейросетей
\Data\Train - данные для тренировки
\Stats - некоторые данные обучения и проверки
------------------------------------------------------------------------
В самом левом окне находится панель управления нейросетями. 
Можно создать новую("Create new"), удалить загруженную("Delete"), 
открыть из файла существующую("Open existing"). Список загруженных
сетей находится в белом окне. 
*Формат данных, в которых сохраняется нейросеть - .json
------------------------------------------------------------------------
В центре указываются параметры тренировки и тестирования. 
Можно открыть данные для тренировки\тестирования("Open training\
testing data"), выбрать степень обучения(learning rate), а также 
выбрать размер эпохи("Epoch size"), количество эпох("Number of epochs") и 
количество примеров в тестировании.
*формат данных тренировки и тестирования - .json
*каждую эпоху данные случайным образом выбираются во время обучения(нет повторения)
------------------------------------------------------------------------
В правой области расположены данные тренировки и тестирования. Они проходят
одновременно, т.е. в одной эпохе сначала происходит обучение, а затем тестирование.
Прогресс эпохи показывает прогресс-бар "Epoch progress", прогресс тренировки - 
"Training progress". Тренировку можно оставновить кнопкой "Stop training", но 
текущая эпоха все ровно завершится. Показывается также текущее время тренировки
("Total training time") и среднее время одной эпохи ("Average epoch time"). Также
показываются номер("Best epoch") и результат("Best result") лучшей эпохи, 
номер("Epoch") и результат("Result") текущей эпохи. Можно затем сохранить нейронную 
сеть ("Save network (best result)" и "Save network (best result)"). Кнопка 
"Show statistics" открывает два графика, на одном из них интерпретация чисел 
нейронной сетью, на другом - прогресс результата с количеством эпох. Кнопка 
"Open query file" позволяет выбрать картинку и посмотреть, как она выглядит в сжатом
состоянии, а также значение выходных нейронов сети на данной картинке.
*сохранять нейросети следует дописывая в конец ".json"
*"Show statistics" и "Open query file" можно активировать и без тренировки
